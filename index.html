Over mij als developer

Ik ben een gemotiveerde student software development met veel interesse in webdesign, front-end en het bouwen van gebruiksvriendelijke websites.

Mohammed Hmydou

Ik ben een student Software Developer aan ROC Mondriaan in Den Haag, gedreven door een passie voor technologie en het oplossen van problemen. Van het maken van interactieve webapplicaties tot het ontwikkelen van mobile applications, ik vind het leuk om ideeën om te zetten in werkelijkheid met code.

Met ervaring in webontwikkeling (HTML, CSS, JavaScript, TypeScript, React, Next.js, Prisma, PHP, MySQL) en mobile application development heb ik meerdere projecten afgerond die zowel mijn technische vaardigheden als mijn creativiteit laten zien.

Mijn doel is om een sterke software developer te worden die betekenisvolle digitale ervaringen creëert. Momenteel ben ik op zoek naar een stageplek waar ik mijn vaardigheden kan toepassen in een professionele omgeving en mezelf verder kan ontwikkelen als developer.
